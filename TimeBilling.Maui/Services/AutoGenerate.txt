Restservice depends on the Refit interface ITimeBillingApi which is autogenerated. 
This interface is generated from the swagger definition of the API. 
The interface is generated by the dotnet tool refitter. 
Install dotnet tool refitter by running:
* dotnet tool install -g refitter

The interface is generated in the TimeBilling.Maui project in the Services folder. 
The interface is generated by running the following command in the solution root:


Generate the interface when the API is running by executing following command in your solution root:

refitter https://localhost:7107/swagger/v1/swagger.json --output ./TimeBilling.Maui/Services/ITimeBillingApi.cs --interface-only --additional-namespace "TimeBilling.Contracts"
